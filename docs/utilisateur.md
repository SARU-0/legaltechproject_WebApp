# Guide Utilisateur - Plateforme LegalTech

Bienvenue sur la documentation utilisateur de la solution de signalement. Ce guide vous aidera à prendre en main l'outil, que vous soyez un collaborateur ou un gestionnaire (RH/Juriste).

## 1. Pour les gestionnaires (RH / Juristes)

### Gestion du Dashboard
Le tableau de bord centralise tous les dossiers entrants.
- **Liste des signalements** : Affiche le titre, la date, l'auteur (ou "Anonyme") et le statut actuel.
- **Assignation** : Vous pouvez vous assigner un dossier ou l'assigner à un autre membre de l'équipe spécialisé.

### Traitement d'un dossier
1. Cliquez sur un dossier pour voir le détail.
2. **Consultation des preuves** : Téléchargez les documents joints de manière sécurisée.
3. **Modifications** : Changez le statut du dossier (Pris en charge, En cours, Résolu).
4. **Dialogue** : Utilisez la section "Commentaires" pour échanger avec le lanceur d'alerte (même s'il est anonyme).

---

## 2. Première Connexion (RH/Juriste)

Si vous avez été invité en tant que membre du personnel :
1. Connectez-vous avec vos identifiants temporaires reçus par email.
2. Une page de configuration s'affichera automatiquement.
3. **Choisissez un pseudonyme** : Il sera utilisé pour vos interactions sur la plateforme.
4. **Définissez votre mot de passe définitif** : Assurez-vous d'utiliser un mot de passe robuste.
5. Une fois validé, vous aurez accès à vos outils de gestion.
